1. Install AWS SDK
npm install @aws-sdk/client-translate

2. Set up AWS Credentials

Make sure you have your AWS_ACCESS_KEY_ID and AWS_SECRET_ACCESS_KEY set up.
You can do this via:

export AWS_ACCESS_KEY_ID=your_key
export AWS_SECRET_ACCESS_KEY=your_secret
export AWS_REGION=your_region


Or use the ~/.aws/credentials file (set via aws configure).

3. Basic Example in Node.js
import { TranslateClient, TranslateTextCommand } from "@aws-sdk/client-translate";

// create client
const client = new TranslateClient({ region: "us-east-1" });

async function translateText() {
  const params = {
    Text: "Hello, how are you?",
    SourceLanguageCode: "en",  // auto works too
    TargetLanguageCode: "es",  // Spanish
  };

  try {
    const command = new TranslateTextCommand(params);
    const response = await client.send(command);
    console.log("Translated text:", response.TranslatedText);
  } catch (err) {
    console.error("Error:", err);
  }
}

translateText();

4. Supported Languages

Use SourceLanguageCode: "auto" if you want Amazon Translate to detect the input language automatically.

For TargetLanguageCode, pass ISO language codes (e.g., fr, de, bn).

5. Example with Auto Language Detection
const params = {
  Text: "Bonjour tout le monde",
  SourceLanguageCode: "auto",
  TargetLanguageCode: "en",
};
